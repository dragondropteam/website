DragonDrop Installation Instructions

Windows

Click on Download to go to the host site for DragonDrop application files.

Click on DragonDrop-Setup-version.exe to download the Windows installer for the selected DragonDrop Version.

Once it finishes downloading, click on the arrow next to 'Discard', and select 'Keep'. We promise DragonDrop is not a virus!

Find the file in the directory it downloaded to, and double click on it.

Click on 'More Info'.

Click on 'Run Anyway'.

Read the License Agreement and click "I Agree" and let it install the files.

Select"Run DragonDrop" if you want the application to launch right away. Click on finish.

This window will pop up when you launch DragonDrop. Enjoy!

macOS

Click on Download to go to the host site for DragonDrop application files.

Click on Dragon.Drop-version.dmg to download the Mac installer for the selected DragonDrop Version.

Right click on the .dmg file and click "Open".

Note: Double-clicking will not yield the same results, so you must right click it to open it.


This window will pop up. Click on Open.

Click on Agree & wait for it to load.

Drag the DragonDrop application to the Applications folder.

Enter your password if prompted.

Go to your Applications folder and double click on the DragonDrop icon.

Click on Open.

Step 8

DragonDrop is ready to use. Enjoy!

Linux

Download the correct DragonDrop for your operating system.

Open the terminal in the same directory as the installer and run 'chmod +x DragonDrop-2.0.0-x86_64.AppImage'

Double click on the AppImage file.

Once it finishes installing the files in the background, DragonDrop will open and be ready to use. Enjoy!


If you are doing anything with Wink, Ringo, or Arduino, please follow these instructions from the Arduino website to install the Arduino IDE before programming the hardware!
